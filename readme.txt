== Description ==

WP-Blogtoppen er et lille plugin, som embedder tracking JavaScriptet fra den danske blograngliste, Blogtoppen.dk.

JavaScriptet bliver inkluderet på alle offentlige sider for alle andre end administratorer.

Plugin'et er ikke udviklet eller vedligeholdt af Blogtoppen.dk.

== Installation ==

1. Upload `wp-blogtoppen.php` til `/wp-content/plugins/` directory'et
2. Aktiver pluginet i 'Plugins'-menuen i WordPress
3. Under 'WP-Blogtoppen' i 'Indstillinger'-menuen indsættes Blogtoppen nøglen (find den i mailen, du har modtaget fra Blogtoppen)
